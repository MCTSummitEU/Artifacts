# Sasha Kranjac

<img src="https://cdn.runevents.net/speaker-profile-images/e06cf865-e869-4f8e-8e30-10043d3c485a" alt="Sasha Kranjac" style="max-width:300px; max-height:300px; width:auto; height:auto;" />

Sasha is a Microsoft Regional Director, Microsoft Security MVP and Microsoft Azure MVP, CEO at Kloudatech, and CEO at Kranjac - IT Training and Consulting.
Sasha and his companies empower professionals and organisations worldwide to learn, be effective and stay secure in the cloud.
Aside from Cloud/Security Architecture and, consulting, they deliver Microsoft, EC-Council, CompTIA and their own custom courses and PowerClass Workshops internationally.
Sasha is also Microsoft Certified Trainer (MCT), MCT Regional Lead, Certified EC-Council Instructor (CEI), CompTIA Instructor, frequent speaker at various international conferences, user groups and events, book author on cloud security, Microsoft Azure, Microsoft 365, and Windows Server.
