# Creating Engaging and Interactive Power BI Training: Balancing Interactivity with Learning Objectives

**Speaker(s):** Gbenga Ayelabola — session on the [European MCT Summit 2025, Manchester](https://e.runevents.net/european-mct-summit-2025) agenda.

## Abstract

Delivering Power BI training that is both highly engaging and structured can be challenging, especially when trying to balance interactivity with achieving the session’s learning objectives. This session addresses the common problem many MCTs face: how to avoid static, webinar-style sessions while ensuring that learners remain focused on the core objectives of the course.

In this session, attendees will explore strategies for transforming Power BI training into interactive, learner-driven experiences. We’ll discuss practical methods such as scaffolding techniques, using real-time feedback to tailor sessions, and maintaining alignment with the core objectives. The focus will be on course delivery, offering actionable insights into how to prevent derailing during interactive training while keeping the session productive and engaging.

Participants will also gain insights into balancing audience expectations with set objectives, ensuring both enjoyment and effectiveness. By incorporating pre- and post-MCT experience, this session will offer a rich set of tools and techniques for delivering training that is both dynamic and outcome-oriented.

Key Takeaways:

	•	Proven techniques for transforming Power BI training sessions into interactive, engaging experiences.
	•	Tools for incorporating scaffolding and in-session feedback without derailing from objectives.
	•	Best practices for balancing audience expectations with streamlined learning goals.

This session is designed to help MCTs elevate their Power BI training delivery by focusing on learner engagement and adaptable, feedback-driven strategies.
