# Deliver an Azure class with the help of advanced PowerShell scripts

**Speaker(s):** Rolf McLaughlin — session on the [European MCT Summit 2025, Manchester](https://e.runevents.net/european-mct-summit-2025) agenda.

## Abstract

Are you tired of being the trainer who wastes the students' time by typing scripts during sessions? Let PowerShell help you deliver an Azure class and also learn how to write advanced PowerShell scripts.
PowerShell is easy, right? Now let's up the game from "one line at a time" and add advanced functionalities, logic, logging and subroutines as well as graphical user interfaces to the mix. And after that make good use of the result in the classroom to focus on delivery and not on typing, typing, typing...
