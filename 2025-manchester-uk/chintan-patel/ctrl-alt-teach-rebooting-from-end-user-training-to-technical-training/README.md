# Ctrl + Alt + Teach: Rebooting from End User Training to Technical Training

**Speaker(s):** Chintan Patel — session on the [European MCT Summit 2025, Manchester](https://e.runevents.net/european-mct-summit-2025) agenda.

## Abstract

In the dynamic landscape of Microsoft technologies, trainers often need to "reboot" their approach to keep pace with evolving demands. Transitioning from teaching end-user courses like Microsoft Office to delivering technical training on platforms such as Azure or Power BI is a significant shift. This session, "Ctrl + Alt + Teach: Rebooting from End User Training to Technical Training," is designed to help trainers navigate this transformation smoothly and effectively.

By pressing "Ctrl + Alt + Teach," we symbolically initiate a reset—altering our control (Ctrl), changing our approach (Alt), and enhancing our teaching (Teach). This session addresses the complexities of this transition, offering strategies to overcome challenges and excel in delivering technical courses.
