# Using tools, best practices, and AI to make your Web site and courseware accessible

**Speaker(s):** Ronald Beekelaar — session on the [European MCT Summit 2025, Manchester](https://e.runevents.net/european-mct-summit-2025) agenda.

## Abstract

As diversity and inclusion become essential for organizations, it is important to understand how to make Web sites and courseware sites accessible for learners with visual, auditory, and mobility disabilities. Microsoft also requires certain accessibility standards from vendor's Web sites. In this session, MVP and MCT Ronald Beekelaar will explain what tools to use to make a Web site accessible according to WCAG guidelines. The session will also cover examples and explore how AI will greatly simplify this task.
