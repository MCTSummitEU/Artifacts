# Ronald Beekelaar

Ronald Beekelaar is a long-time MCT and also Hyper-V MVP, and founder of Virsoft Solutions. Virsoft is an Authorized Lab Hoster with Microsoft, and provides access to thousands of hosted hands-on-lab and demo environments for MCTs, training centers, Microsoft events, and other customers. Ronald is a well-known presenter on the topic of virtualization, Hyper-V, security and Azure.
