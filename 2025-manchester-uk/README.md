# 2025 Manchester — speaker artifacts

Session tree built from [run.events](https://e.runevents.net/european-mct-summit-2025) (**European MCT Summit 2025**). Each leaf folder is one agenda session; `README.md` contains the title, speakers, and abstract from the API.

## Naming

- **Speaker folder:** for a single speaker, `givenname-surname` (kebab-case). For multiple speakers on the same session, each person as `givenname-surname`, chain sorted **alphabetically** by that slug (e.g. `bill-ayers-sasha-kranjac`).

- **Session folder:** kebab-case slug from the session title.

- **Same person, different sessions:** different folders when the **set of speakers** differs (e.g. solo vs panel).

See also the [2026 Sarajevo](../2026-sarajevo-ba/README.md) tree for the full convention (including `_` prefixes for placeholders).
