# Using GitHub Copilot to Build Games in Visual Studio Code

**Speaker(s):** Engin Soysal — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

Can GitHub Copilot be used creatively, not just for business apps, but for games? In this session, we explore how GitHub Copilot can assist in building simple games inside Visual Studio Code.

Using practical examples, I’ll show how Copilot helps with game logic, state handling, mechanics, and iteration speed, while also highlighting its limitations. We’ll discuss prompt strategies, creativity versus control, and how Copilot fits into experimental or educational game development.

This session is ideal for developers who want to explore Copilot beyond standard CRUD applications and see how AI can support creative coding without replacing developer intent.
