# Flexible Power BI trainings - getting ready to personalised trainings on-scale

**Speaker(s):** Dmitrii Solovev — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

Scaling Power BI training has always been a challenge. When you teach large groups, reviewing lab work and giving meaningful feedback quickly becomes a bottleneck. Until recently, the only realistic options were automated tests or very limited manual checks—neither of which really help learners understand where they went wrong.

With Power BI MCP, the new PBIP + PBIR project format, and Azure AI Foundry, this is finally changing. These tools make it possible to build AI-assisted lab reviews that can analyze real Power BI projects, identify mistakes, and guide learners toward the exact skills they need to improve—at scale.

Agenda:
1. Introduction
2. PBIP and PBIR vs. traditional PBIX files
3. A quick overview of Azure AI Foundry
4. Building an agent to analyze PBIP projects
5. Building an agent that assesses knowledge gaps and suggests a personalized learning path
