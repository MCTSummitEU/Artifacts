# How AI changes Power BI and Fabric development world and what it means for trainers

**Speaker(s):** Dmitrii Solovev — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

The goal of the session to cover significant changes came to Power BI and Fabric world with the most recent LLMs and how this affects the trainings. Session agenda:
1. Demo - creating Power BI report using Github Copilot from a scratch.
2. Demo - Power BI copilot on AI-prepped vs. not prepped model
3. Why data modelling knowledge is critical for AI-augmented development and maintenance?
4. Where must and avoid using AI-tools in Power BI
5. Main risks and limitations
