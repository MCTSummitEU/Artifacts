# Securing the Core: Building a Rock-Solid Microsoft 365 Tenant Baseline

**Speaker(s):** Nicky De Westelinck — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

In today’s cloud-first world, Microsoft 365 is the backbone of productivity for millions of organizations, but with great power comes great responsibility. This session dives into how to build a secure, resilient Microsoft 365 tenant by leveraging proven security best practices to build a secure Microsoft 365 Foundation.

You’ll learn how to establish a strong security baseline that protects your core services, Exchange Online, SharePoint, Teams, and Entra ID, against evolving threats. We’ll explore practical steps to harden configurations, enforce identity protections, and reduce the attack surface, all while maintaining usability and compliance. Whether you're just starting your Microsoft 365 journey or looking to reinforce your existing setup, this session will equip you with actionable insights to secure your environment from the ground up.
