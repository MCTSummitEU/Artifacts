# From Workflows to AI Agents: Building Intelligent Automation with Power Automate & Copilot Studio

**Speaker(s):** Thomas Janssen — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

In this session you will learn how to create real business value by combining workflows and AI agents using Microsoft Power Automate and Copilot Studio.

Key topics:

- How to harness tangible business value from AI, beyond experimentation and hype

- Introduction to Copilot Studio: what AI agents are and how they fit in the Microsoft ecosystem

- Designing and building workflows with Power Automate as the foundation of automation

- Workflows vs. AI agents: clear guidelines on when to use Power Automate, when to use agents, and when to combine both

- Build-with-me demo: creating an intelligent solution by connecting workflows and AI agents in practice

Takeaways:

- A practical mental model for workflows and agents

- Clear decision criteria for choosing the right automation approach

- A reusable pattern you can apply in your own organization
