# The Oversharing Solution Blueprint

**Speaker(s):** Sara Fennah — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

Did you know Microsoft have a Microsoft 365 Copilot Adoption blueprint to prevent oversharing? The proposed solution uses tools from across Microsoft 365 including SharePoint Online, SharePoint Advanced Management, Microsoft Teams and Microsoft Purview to address oversharing and protect data from accidental discovery with Microsoft 365 Copilot. In this session we'll look into the solutions proposed considering their various benefits and challenges to create good, better and best solutions to managing oversharing in Microsoft 365.

This session will include the following technologies and features:

SharePoint Advanced Management
Access Reviews
Restricted SharePoint Search
Restricted Content Discovery
Sensitivity Labels
Data Loss Prevention
Adaptive Protection
Data Lifecycle Management
Communication Compliance
DSPM
You'll leave the session with:

An understating of Microsoft Oversharing Blueprint
Knowledge of options available to manage information available to Microsoft 365 Copilot
Practical tips to increase data security for Microsoft 365.
