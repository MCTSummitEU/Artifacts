# How to Modernize Identity and Access in Hybrid Environments

**Speaker(s):** Nedim Sadovic — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

Modernizing identity and access is essential for securing organizations operating in hybrid environments. Traditional perimeter-based models are no longer sufficient, shifting the focus toward Zero Trust principles and centralized identity management through Microsoft Entra ID. This session explores how to consolidate identities, implement MFA and Conditional Access, automate user lifecycle processes, and integrate on‑premises Active Directory with cloud services. Through practical examples, it will demonstrate how a modern IAM approach reduces risk, simplifies operations, and enables a secure, scalable, and agile infrastructure.
