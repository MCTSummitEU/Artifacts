# Microsoft Dynamics Business Central + Microsoft Foundry: Practical AI models and agents for real ERP work

**Speaker(s):** Nebojsa Visic — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

This talk is a field guide to building private, governed AI inside Microsoft Dynamics 365 Business Central using Microsoft Foundry, no third-party model sprawl, full enterprise control. We’ll cover the end-to-end path: selecting and evaluating models, wiring secure inference from AL/Azure Functions, orchestrating tool-use and retrieval, and keeping costs predictable. Through live mini-demos (agent-style enrichment, data validation, and smart suggestions inside BC flows), you’ll learn patterns that actually ship: request/response contracts, throttling, fallbacks, red-teaming, and telemetry. We’ll also map what to do when requirements evolve, from “assist” to fully agentic workflows while staying within Microsoft’s AI platform. You’ll leave with reference patterns and a checklist to stand up your first Foundry-powered capability in days, not months.
