# Vibe or Not to Vibe Code? The Question Is Now!

**Speaker(s):** Nebojsa Visic — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

AI is no longer just a feature of modern platforms, it has become a daily tool for developers working across all generations of ERP systems.
In this session, we explore how AI-assisted and “vibe coding” approaches can dramatically improve productivity, code quality, and long-term relevance for developers working with Microsoft Dynamics NAV and Dynamics 365 Business Central, including legacy environments such as Navision Financials.
Based on real-world projects and hands-on experience, the talk shows how modern AI tools (Copilot-style assistants, private/open LLMs, prompt libraries, and agent-based workflows) can be effectively integrated into everyday ERP development, even when the target system itself hasn’t changed in years. 
From classic object designers to VS Code, centric workflows, AI becomes the missing bridge between old platforms and modern development practices.

The focus is not on chasing the latest ERP trend, but on empowering developers:
how to use AI responsibly, productively, and pragmatically to stay efficient, adaptable, and future-proof, regardless of the ERP version or tooling limitations.
This is a practical, experience-driven session for developers who want to work smarter today and remain relevant tomorrow, with AI as a co-pilot, not a replacement.
