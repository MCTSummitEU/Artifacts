# Prompt to Podium: Automating Demos with VS Code & Copilot

**Speaker(s):** Saul Patino — session on the [MCT EU Summit 2026](https://e.runevents.net/mct-eu-summit-2026) agenda.

## Abstract

For the modern Microsoft Certified Trainer (MCT) and technical lead, preparation time is often the scarcest resource. This session is designed for intermediate-level trainers and developers who possess a foundational understanding of Azure and source control but want to dramatically accelerate their workflow. We will move beyond basic coding to explore a fully integrated ecosystem that turns ideas into deployable classroom scenarios instantly.

Attendees will experience a live, end-to-end workflow demonstrating how to:

Authenticate and Connect: Seamlessly tie Azure accounts and GitHub credentials into VS Code for a unified control plane.

Generate with AI: Leverage GitHub Copilot to scaffold infrastructure code and application logic simultaneously.

Automate Documentation: Use GitHub capabilities to auto-generate "Wiki" style guides and student instructions alongside the code.

Persist and Share: Store session artifacts in GitHub Repositories for immediate distribution and future reuse.

Expect a highly practical, demo-heavy presentation that prioritizes "doing" over "slides," leaving you with a concrete blueprint to modernize your lab creation process.
